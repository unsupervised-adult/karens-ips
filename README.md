# Karen's IPS

An intelligent Intrusion Prevention System for family-friendly network security.

## Overview

Karen's IPS is a family-oriented network security solution that provides intelligent content filtering, threat detection, and network monitoring capabilities. Built with Python and machine learning models, it offers real-time protection while maintaining ease of use for home and family networks.

## Project Structure

- `src/` - Python modules and core functionality
- `models/` - TensorFlow Lite models for ML-based detection
- `training/` - Data collection and model training scripts
- `deployment/` - Installation and deployment scripts
- `scripts/` - Utility scripts
- `tests/` - Unit tests
- `config/` - YAML configuration files

## Features

- Real-time network monitoring
- ML-based threat detection
- Family-friendly content filtering
- Easy installation and configuration
- Lightweight deployment

## Installation

See the `deployment/` directory for installation scripts and documentation.

## Configuration

Configuration files are stored in the `config/` directory as YAML files.

## Testing

Run tests from the project root:
```bash
python -m pytest tests/
```
